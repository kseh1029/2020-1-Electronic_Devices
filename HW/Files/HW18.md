# HW18

전자소자 (김학린)

HW#18 (05/13, 수요일) - (제출마감일 : 5/19 화요일)

1. MOSFET의 high frequency operation에서 low frequency operation과 달리 MOSFET input 단의 gate capacitance와 input resistance가 output current level에 크게 영향을 줄 수 있음을 설명해 보시오.
